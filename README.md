# JOBLY
Jobly is a modern, accessible platform that helps job seekers build standout CVs, discover relevant roles, and track their application journey with intelligent, skills-based matching

# Core features:

  CV builder: Create a professional CV with structured sections, live preview, and export to PDF.
  
  Professional templates: Choose from polished, ATS-friendly templates optimized for readability and impact.
  
  Job listings: Browse curated roles with robust filters (title, location, salary, remote) and saved searches.
  
  Application tracker: Log applications, update statuses, add notes, and keep deadlines visible.
  
  Skills training: Access targeted modules and resources to close skill gaps aligned to job requirements.
  
  Mentorships page: Connect with mentors, view profiles, and request guidance tailored to your goals.
  
  Interview tips: Get actionable advice, question banks, and checklists for structured preparation.
  
  Job matching: Match roles to your CV and see a compatibility percentage based on skills and experience.
  
  Accessibility & responsiveness: Inclusive design, keyboard navigation, and a mobile-first responsive UI.

# Tech stack:

  Frontend: HTML, CSS, JavaScript (responsive UI/UX with accessibility-first patterns)
  
  Backend & data: Firebase (auth, Firestore/Realtime DB), optional serverless functions for secure API calls
  
  Email & notifications: EmailJS (transactional flows), with backend proxy to protect keys
  
  Deployment: Netlify (CI builds, environment variables, custom domain + DNS)
  
  Analytics & monitoring: Optional: Google Analytics / Plausible, Netlify logs
